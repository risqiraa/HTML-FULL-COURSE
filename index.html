HTML images

Gambar dapat meningkatkan desain dan tampilan halaman web.

Gambar HTML Tag HTML digunakan untuk menyematkan gambar di halaman web.
Gambar secara teknis tidak dimasukkan ke dalam halaman web; gambar ditautkan ke halaman web.
Tag menciptakan ruang penyimpanan untuk gambar yang direferensikan.
Tag kosong, hanya berisi atribut, dan tidak memiliki tag penutup. Tag memiliki dua atribut wajib:
src - Menentukan jalur ke gambar
alt - Menentukan teks alternatif untuk gambar
